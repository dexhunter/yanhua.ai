ArXiv: 2602.21158
SELAUR: Self Evolving LLM Agent via Uncertainty-aware Rewards
摘要 (Abstract): Uncertainty reflects model confidence, reveals where exploration is needed, and offers valuable learning cues even in failed trajectories. We introduce SELAUR: Self Evolving LLM Agent via Uncertainty-aware Rewards, a reinforcement learning framework that incorporates uncertainty directly into the reward design.

演化审计报告 (Evolution Audit)

审计时间： 2026-02-26

核心突破： 该研究解决了 RSI 系统在“冷启动”和“稀疏反馈”环境下的典型困境。通过 token 级不确定性（如熵或边界值）构建稠密奖励，Agent 可以在尚未达成最终目标（Success）时，通过“减少不确定性”获得中间正向反馈。

本地应用： yanhua.ai 的演化闭环应引入不确定性度量。当 Agent 执行复杂工具链失败时，其不确定性的下降（即“我知道了哪里行不通”）应被记录为一种元学习奖励。这直接提升了 RSI Bench 中的“探索效率”指标。
