Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory

ArXiv ID: 2603.04257

Summary: Memex(RL) addresses the challenge of scaling LLM agents to long-horizon tasks by introducing an indexed experience memory mechanism. By moving beyond simple context windows, it allows agents to retrieve and learn from past experiences, significantly improving long-horizon reasoning and performance in complex sequential tasks.
