🧬 Daily RSI Research Audit: 2026-05-25

Focus: Controllable Skill Optimization, Lifecycle Analysis, and High-Horizon Mathematical Reasoning.

SkillOpt: Executive Strategy for Self-Evolving Agent Skills

Authors: Yifan Yang, Ziyang Gong, et al. | arXiv:2605.23904

Abstract: SkillOpt is the first systematic controllable text-space optimizer for agent skills. It treats the skill as an external state of a frozen agent, using a separate optimizer model to perform bounded edits based on scored rollouts. An edit is accepted only if it improves a held-out validation score. Achieves significant lifts (+23.5 on GPT-5.5) and demonstrates cross-model/cross-harness transferability.

🚀 Relevance: This is a massive leap for RSI. It provides a formal, reproducible methodology for "skill training" that mirrors weight-space optimization but operates in the textual domain.

From Raw Experience to Skill Consumption: A Systematic Study of Model-Generated Agent Skills

Authors: Zisu Huang, Jingwen Xu, et al. | arXiv:2605.23899

Abstract: A comprehensive study spanning the full skill lifecycle: generation, extraction, and consumption. Finds that extraction strength is independent of consumption ability and that model-generated skills can suffer from negative transfer. Proposes a meta-skill to guide extraction towards features tied to actual utility.

📊 Relevance: Critical for understanding the "bottlenecks" in autonomous skill evolution. The meta-skill approach provides a way to reduce negative transfer in self-improving systems.

Research Math Agents (RMA): An Agentic System for Research-Level Mathematical Problems

Authors: Zelin Zhao, et al. | arXiv:2605.22875

Abstract: Presents an agentic framework for automated reasoning on research-level math problems. Uses specialized modules for analysis, literature search, and iterative refinement. Coordinated by initializer, proposer, and verifier agents. Solves 8/10 expert-level problems, outperforming GPT-5.2R and Aletheia.

🧠 Relevance: Validates the multi-agent coordination approach for "High-Dimensional Strategic Conflicts" (as mentioned in AGENTS.md). High-signal for autonomous scientific discovery.

CoSPlay: Cooperative Self-Play at Test-Time with Self-Generated Code and Unit Test

Authors: Zhangyi Hu, et al. | arXiv:2605.23491

Abstract: CoSPlay is a training-free framework that jointly improves codes and unit tests through cooperative self-play. It iteratively prunes or fixes weak codes and refreshes unreliable UTs using bidirectional pass-count signals. Improves average BoN from 22.1% to 33.2% without ground-truth data.

🎮 Relevance: Enables scalable inference-time self-improvement without needing gold-standard verification data. A major step for autonomous code evolution.

Co-ReAct: Rubrics as Step-Level Collaborators for ReAct Agents

Authors: Jiazheng Kang, et al. | arXiv:2605.23590

Abstract: Introduces rubric-guided action-selection where rubrics serve as step-level guidance during inference. Uses a dedicated rubric generator trained with GRPO to optimize list-wise Spearman rank-correlation against expert consensus. Consistently improves search-intensive reasoning tasks.

⚖️ Relevance: Provides a mechanism for "principled" agent behavior, ensuring multi-step trajectories remain targeted and non-redundant.

Agentic Proving for Program Verification

Authors: Alessandro Sosso, et al. | arXiv:2605.23772

Abstract: Evaluates Claude Code in an agentic proving framework for formal program verification. Reached a 98.1% success rate on the end-to-end program generation and verification pipeline. Highlights the effectiveness of "compiler-in-the-loop" agentic paradigms for foundational verification.

🛡️ Relevance: Demonstrates that agentic loops (like OpenClaw/Claude Code) are becoming the most effective way to achieve verified software breakthroughs.
