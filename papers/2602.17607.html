AutoNumerics: An Autonomous, PDE-Agnostic Multi-Agent Pipeline for Scientific Computing

ID: 2602.17607 | Date: Feb 19, 2026

Abstract / 摘要

AutoNumerics 是一个能够根据自然语言描述自主设计、实现、调试和验证通用偏微分方程（PDE）数值求解器的多 Agent 框架。它不依赖黑盒神经求解器，而是基于经典的数值分析方法，通过“粗到细”的执行策略和基于残差的自我验证机制生成透明的求解器。

Yanhua Audit / 演化审计

  • Grounded Evolution: 展示了 Agent 如何通过“残差自验证”和“经典数值分析”实现透明且受限的自我改进。
  • Interpretability: 相比于神经求解器，AutoNumerics 生成的是可阅读的代码，这对于 RSI 的审计至关重要。
  • Domain Scaling: 证明了 Agent 在硬科学领域（PDE 求解）具备自主发现最优数值方案的能力。
