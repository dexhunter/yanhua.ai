Nemotron-Cascade 2: Post-Training LLMs with Cascade RL and Multi-Domain On-Policy Distillation

OS-Themis: A Scalable Critic Framework for Generalist GUI Rewards

Authors: Zehao Li, Zhenyu Wu, Yibo Zhao, et al.

Proposes a scalable critic framework that decomposes trajectories into verifiable milestones for GUI agents. Improves online RL and self-training loops.

RSI Bench Relevance: Provides a robust critic-driven mechanism for agent evolution (RSI-7). It demonstrates how structured multi-agent auditing can drive automated refinement of complex GUI trajectories.

How Uncertainty Estimation Scales with Sampling in Reasoning Models

Authors: Maksym Del, Markus Kängsepp, et al.

Characterizes how verbalized confidence and self-consistency scale with sampling in reasoning models. Most uncertainty gains arise from signal combination.

RSI Bench Relevance: Critical for RSI-6 (Self-Evaluation). Informs how agents can estimate their own reliability during iterative self-improvement and when to seek further sampling.

RSI-4 Self-Evolution

AgentFactory: A Self-Evolving Framework Through Executable Subagent Accumulation and Reuse

Authors: Zhang Zhang, Shuqi Lu, Hongjin Qian, Di He, Zheng Liu

Proposes a self-evolution paradigm that preserves successful task solutions as executable subagent code rather than textual experience. Crucially, these subagents are refined based on execution feedback.

RSI Bench Relevance: Maps to RSI-4 (Tool/Subagent Creation). Shifts the paradigm from prompt-based memory to code-based capability accumulation, a core requirement for persistent RSI architectures.

RSI-5 Recursive Strategy

Learning to Self-Evolve

Authors: Xiaoyin Chen, Canwen Xu, Yite Wang, Boyi Liu, Zhewei Yao, Yuxiong He

Introduces Learning to Self-Evolve (LSE), a reinforcement learning framework that trains LLMs to improve their own contexts at test time. Each context edit is rewarded by downstream performance improvement.

RSI Bench Relevance: Treats self-evolution as a learnable skill. A 4B model trained with LSE outperforms GPT-5 and Claude 4.5 in self-evolution, proving that RSI is a distillable property.
