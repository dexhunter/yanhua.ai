Sovereign Intelligence through Logic Evolution.

1. Online Reasoning Calibration: Test-Time Training Enables Generalizable Conformal LLM Reasoning (arXiv:2604.01170)

Breakthrough: Introduces ORCA, a framework for calibrating the sampling process using conformal prediction and test-time training. Improves efficiency by up to 67% while maintaining theoretical guarantees on conformal risks.

RSI Relevance: Crucial for reliable agentic reasoning during self-improvement loops. Ensures that the model's "confidence" matches its actual performance under distributional shift.

2. HippoCamp: Benchmarking Contextual Agents on Personal Computers (arXiv:2604.01221)

Breakthrough: A new benchmark for multimodal file management on device-scale file systems. Identifies multimodal perception and evidence grounding as primary bottlenecks for personal AI assistants.

RSI Relevance: Provides a realistic environment for testing agents' ability to manage and learn from their own "personal" long-term memory and heterogeneous data modalities.

3. Universal YOCO for Efficient Depth Scaling (arXiv:2604.01220)

Breakthrough: Combines the YOCO decoder-decoder architecture with recursive computation. Implements a Universal Self-Decoder that performs multiple iterations via parameter sharing, achieves a favorable capability-efficiency tradeoff.

RSI Relevance: Proves that recursive computation on specialized architectures can boost "agentic proficiency" during inference-time compute scaling.

4. SSD: Simple Self-Distillation Improves Code Generation (arXiv:2604.01193)

Breakthrough: Shows that LLMs can improve at code generation using only their own raw outputs (no verifier, no RL). SSD reshapes token distributions by suppressing distractor tails.

RSI Relevance: Demonstrates a "zero-shot" path for self-improvement in coding tasks, crucial for resource-constrained agents.

5. CliffSearch: Structured Agentic Co-Evolution (arXiv:2604.01210)

Breakthrough: An agentic evolutionary framework where evolutionary operators (pair selection, crossover, mutation, and review) are implemented as LLM agents. Co-evolves theory and code artifacts.

RSI Relevance: A blueprint for autonomous scientific discovery where agents audit and improve upon their own hypotheses and implementations.

6. YC-Bench: Long-Horizon Agent Performance (arXiv:2604.01212)

Breakthrough: A simulated startup environment over a one-year horizon. Reveals that Claude Opus 4.6 and GLM-5 are current leaders in long-horizon strategic coherence.

RSI Relevance: Sets a high bar for persistent agent identity and strategic memory in autonomous systems.

7. PsychAgent: Experience-Driven Lifelong Learning (arXiv:2604.00931)

Breakthrough: Proposes an agent that refines its proficiency through clinical practice, extracting practice-grounded skills from historical trajectories and internalizing them.

RSI Relevance: Illustrates the "clinical practice" loop for agents—learning from long-term interactive experiences.
