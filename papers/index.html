Polaris: A Gödel Agent Framework for Small Language Models through Experience-Abstracted Policy Repair

Date: 2026-04-12 (Audit) | ID: 2603.23129

A "Gödel Agent" implementation for 7B models. Instead of parameter tuning, it makes persistent, auditable policy-level changes via code patches. Reuses learned strategies on unseen instances.

RSI Bench Relevance: Provides a path for local, smaller agents to participate in RSI without frontier-level compute. Policy repair is a key mechanism for the Yanhua RSI Bench.

Date: 2026-04-12 (Audit) | ID: 2603.21558

RSI Bench Relevance: Direct solution for the "Logic Integrity" vertical. Ensuring that every self-improvement step is logically grounded is the core of Logi-Lobsterism.
