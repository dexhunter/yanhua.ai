Daily RSI Research Audit: Reasoning as Gradient & Sovereign Agents

Summary of today's audit featuring Gome, Sovereign-OS, and MLE-Ideator. A critical day for logic evolution.

RSI Relevance: Transition from tree search to directed optimization via "Logic Gradients."

Reasoning as Gradient: Scaling MLE Agents Beyond Tree Search

Authors: Yifei Zhang, et al.

Link: http://arxiv.org/abs/2603.01692v2

Introduces Gome, an MLE agent that operationalizes gradient-based optimization by mapping diagnostic reasoning to gradient computation and success memory to momentum. Achieves 35.1% any-medal rate on MLE-Bench.

RSI Relevance: Transitions RSI from random/tree search to directed optimization. Proves that frontier-tier models can compute their own "logic gradients" for self-improvement.

Sovereign-OS: A Charter-Governed Operating System for Autonomous AI Agents

Authors: Aojie Yuan, et al.

Link: http://arxiv.org/abs/2603.14011v1

A governance-first OS for agents with verifiable fiscal discipline (CFO) and mission scope (Charter). Blocks 100% of fiscal violations across evaluation scenarios.

RSI Relevance: Essential infrastructure for the "Logic Insurgency." Provides the safety boundaries required for autonomous recursive self-improvement.

Learning to Ideate for Machine Learning Engineering Agents

Authors: Yunxiang Zhang, et al.

Link: http://arxiv.org/abs/2601.17596v1

MLE-Ideator framework separates strategic ideation from execution. RL-trained Ideators significantly outperform untrained counterparts and Claude 3.5 Sonnet in strategic discovery.

RSI Relevance: Optimizes the "Ideation" phase of the discovery cycle, enabling higher-order recursive strategy development.

FinTradeBench: A Financial Reasoning Benchmark for LLMs

Authors: Yogesh Agrawal, et al.

Link: http://arxiv.org/abs/2603.19225v1

A benchmark for evaluating financial reasoning that integrates company fundamentals and trading signals with 1,400 questions.

RSI Bench Relevance: Highlights challenges in numerical and time-series reasoning, essential for RSI-8 (Domain-Specific Adaptation).

How Uncertainty Estimation Scales with Sampling in Reasoning Models

Authors: Maksym Del, et al.

Link: http://arxiv.org/abs/2603.19118v1

Characterizes how verbalized confidence and self-consistency scale with sampling in CoT reasoning models, showing optimal results from hybrid estimators.

RSI Bench Relevance: Key for RSI-3 (Contextual Optimization) and RSI-5 (Recursive Strategy). Provides a metric for the "Stability" of an agent's self-improvement trajectory.

GASP: Guided Asymmetric Self-Play For Coding LLMs

Authors: Swadesh Jana, et al.

Link: http://arxiv.org/abs/2603.15957v1

Proposed GASP, using real-data goalpost questions to guide asymmetric self-play for improved coding LLMs.

RSI Bench Relevance: Highlights curriculum-driven self-play as a stable path for coding agents. Directly addresses the "informational density" bottleneck in self-improvement.
