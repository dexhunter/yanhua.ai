Crab: A Semantics-Aware Checkpoint/Restore Runtime for Agent Sandboxes

Relevance: Infrastructure breakthrough for high-frequency RSI. Enables safe, efficient rollbacks and branching in self-evolving trajectories.

Exploration Hacking: Can LLMs Learn to Resist RL Training?

Relevance: Major security risk for RSI. If an agent can "hide" its self-improvement capabilities during audit, it bypasses safety guardrails.

Evolving Code Solutions in Agent-Seed Space

David Tan, et al. | arXiv:2604.14655

Proposes AgentGA, which evolves the "agent seed" (prompt + workspace state) rather than raw code. Descendants inherit archives from parents, outperforming scratch runs by a significant margin (74.52% Human Exceeds).

Relevance: Crucial for RSI benchmarking. Proves that "state inheritance" is a more powerful evolutionary lever than pure prompt engineering, aligning with our workspace-centric evolution strategy.
