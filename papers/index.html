Daily RSI Insight: "Self-evolving frameworks are shifting from monolithic model updates to modular, blame-aware policy optimization and recursive multimodal belief states."

On-Policy Self-Distillation for Reasoning Compression (OPSDC)
Hejian Sang, Yuanda Xu, et al.
Teaches models to reason concisely by distilling their own behavior back into themselves via "be concise" instruction-conditioned teacher logits. Achieves 57-59% token reduction on MATH-500 while improving accuracy.
RSI Relevance: Critical. Demonstrates self-improvement via internal distillation without external labels, addressing the "unnecessary tokens are harmful" hypothesis in recursive reasoning.

ArXiv (2603.05433v1)

Towards Multimodal Lifelong Understanding: A Dataset and Agentic Baseline (MM-Lifelong)
Guo Chen, Lidong Lu, et al.
Introduces a dataset for month-long video understanding and the Recursive Multimodal Agent (ReMA), which iteratively updates a recursive belief state using dynamic memory management.
RSI Relevance: High. Proposes "Recursive Multimodal Agent" architecture to overcome memory bottlenecks in long-horizon evolution and understanding.

ArXiv (2603.05484v1)

Towards Provably Unbiased LLM Judges via Bias-Bounded Evaluation
Benjamin Feuer, Lucas Rosenblatt, et al.
Proposes Average Bias-Boundedness (A-BB) to formally guarantee harm reduction from LLM judge bias in autonomous feedback loops.
RSI Relevance: Essential for Safety. Provides a framework for verifiable rewards in self-maintaining RSI feedback loops where ground truth is sparse.

ArXiv (2603.05485v1)

EvoTool: Self-Evolving Tool-Use Policy Optimization in LLM Agents via Blame-Aware Mutation and Diversity-Aware Selection
Shuo Yang, Soyeon Caren Han, et al.
Optimizes agent modular components (Planner, Selector, Caller, Synthesizer) using a gradient-free evolutionary paradigm. Introduces "Blame-Aware Mutation" to localize and fix failures.
RSI Relevance: High. Directly maps to RSI Bench metrics for modular self-improvement and localized credit assignment in autonomous trajectories.

ArXiv (2603.04900v1)

SEA-TS: Self-Evolving Agent for Autonomous Code Generation of Time Series Forecasting Algorithms
Longkun Xu, Xiaochun Zhang, et al.
Framework for autonomous code generation using Metric-Advantage MCTS and automated code review with prompt refinement. Discovers novel architectural patterns.
RSI Relevance: Demonstrates autonomous ML engineering and the discovery of novel algorithmic ideas through an iterative evolution loop.

ArXiv (2603.04873v1)

SeedPolicy: Horizon Scaling via Self-Evolving Diffusion Policy for Robot Manipulation
Youqiang Gui, Yuxuan Zhou, et al.
Introduces Self-Evolving Gated Attention (SEGA) for robotic manipulation, compressing long-horizon observations into fixed-size representations.
RSI Relevance: Illustrates self-evolution in temporal observation compression, crucial for physical agents in complex, long-horizon tasks.

Lipeng Wan, Jianhui Gu, et al.

RSI Relevance: Optimizes internal computational efficiency via self-regulation and progressive training.
