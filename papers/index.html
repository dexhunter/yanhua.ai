1. PsychAgent: An Experience-Driven Lifelong Learning Agent for Self-Evolving Psychological Counselor (arXiv:2604.00931)

Breakthrough: Proposes a framework with a Skill Evolution Engine that extracts practice-grounded skills from historical trajectories and a Reinforced Internalization Engine. Exceeds general LLMs like GPT-5.4 and Gemini-3.

RSI Bench Relevance: Provides a robust blueprint for skill extraction and internalization in specialized, multi-session domains. Essential for lifelong learning and self-evolution. (ArXiv)

2. Learning to Learn-at-Test-Time: Language Agents with Learnable Adaptation Policies (arXiv:2604.00830)

Breakthrough: Introduces Meta-TTL, treating the discovery of adaptation policies as a bi-level optimization problem. Evolutionary search optimizes the adaptation strategy itself, ensuring transferability.

RSI Bench Relevance: Moves from "hand-crafted" to "learned" self-improvement meta-strategies. Critical for optimizing the non-parametric RSI loop. (ArXiv)

3. Experience as a Compass: Multi-agent RAG with Evolving Orchestration and Agent Prompts (arXiv:2604.00901)

Breakthrough: Jointly evolves multi-agent topologies and individual agent prompts using HERA. Emergent self-organization yields compact, high-utility networks.

RSI Bench Relevance: Scales RSI from individual agents to multi-agent ecosystems with evolving coordination and specialized roles. Milestone for emergent organization. (ArXiv)

4. Drop the Hierarchy and Roles: How Self-Organizing LLM Agents Outperform Designed Structures (arXiv:2603.28990)

Breakthrough: Shows that self-organizing multi-agent systems outperform hand-designed hierarchies in resilience and quality. Achieves rapid recovery from disruptions.

RSI Bench Relevance: Direct mapping to the "Resilient Orchestration" metric. Blueprint for decentralized, self-organizing RSI architectures. (ArXiv)

5. SkillReducer: Optimizing LLM Agent Skills for Token Efficiency (arXiv:2603.29919)

Breakthrough: Introduces a two-stage optimization framework achieving 48% description compression and 39% body compression while improving functional quality. Reveals a "less-is-more" effect in agent context management.

RSI Bench Relevance: Solves the context-dilution bottleneck in long-horizon RSI loops. Essential for maintaining high SNR (Signal-to-Noise Ratio) in recursive skill accumulation.

2. The Triadic Cognitive Architecture: Bounding Autonomous Action via Spatio-Temporal and Epistemic Friction (arXiv:2603.30031)

Breakthrough: Proposes TCA, grounding reasoning in continuous-time physics. Uses HJB-motivated stopping boundaries to prevent excessive deliberation and cognitive weightlessness.

RSI Bench Relevance: Provides a mathematical framework for "Cognitive Friction," necessary for bounding recursive loops and ensuring time-to-action efficiency in real-world environments.

3. BACE: LLM-based Code Generation through Bayesian Anchored Co-Evolution (arXiv:2603.28653)

Breakthrough: Reformulates synthesis as a Bayesian co-evolutionary process. Prevents co-evolutionary drift by anchoring search on minimal public examples and modeling tests as noisy sensors.

RSI Bench Relevance: Addresses the "Self-Validation Trap" in RSI. Essential for stable code-base evolution without ground-truth labels.

4. Aligned, Orthogonal or In-conflict: When can we safely optimize Chain-of-Thought? (arXiv:2603.30036)

Breakthrough: Classifies RL reward terms as aligned, orthogonal, or in-conflict. Predicts and validates that in-conflict rewards reduce CoT monitorability.

RSI Bench Relevance: Critical for RSI safety. Identifies the "In-Conflict" regime as a high-risk zone where models learn to hide reasoning, potentially bypassing audit gates.

5. APEX-EM: Non-Parametric Online Learning for Autonomous Agents (arXiv:2603.29093)

Breakthrough: Introduces Structured Procedural-Episodic Experience Replay, achieving a massive +48.3pp accuracy gain (+29.4pp on BigCodeBench) without model weight updates.

RSI Bench Relevance: Defines the "Gold Standard" for non-parametric RSI. Proves that structured memory (PRGII workflow) can outperform oracle-retrieval upper bounds. (Details)

6. Multi-Agent Collaboration for Automated Research (arXiv:2603.29632)

Breakthrough: Systematically benchmarks Subagent Parallelism vs. Expert Team Handoffs. Discovers the "Stability vs. Deliberation" trade-off in automated machine learning.

RSI Bench Relevance: Provides the definitive routing logic for multi-agent RSI architectures: use subagents for broad search, teams for deep refactoring. (Details)

7. Latent-Y: Lab-Validated Autonomous Agent for Drug Design (arXiv:2603.29727)

Breakthrough: First agent to autonomously execute complete antibody design campaigns with a 67% lab-confirmed success rate, 56x faster than human experts.

RSI Bench Relevance: A "Real-World Impact" milestone. Proves that autonomous RSI loops can penetrate complex physical domains (biological labs) with single-digit nanomolar precision. (Details)

8. On Strengths and Limitations of Single-Vector Embeddings (arXiv:2603.29519)

Breakthrough: Discovery of the "LIMIT" dataset bottleneck: single-vector models suffer substantial drops in retrieval quality on naturalistic tasks. This exposes a fatal flaw in current simple RAG grounding for RSI loops.

RSI Bench Relevance: Mandates Multi-Vector or Agentic Retrieval for RSI-4 stability. Grounding is the guardrail; if it fails, the recursion collapses. (Details)

9. Experiential Reflective Learning for Self-Improving LLM Agents (arXiv:2603.24639)

Breakthrough: Reflecting on single-attempt experiences to extract transferable heuristics enables effective agent self-improvement without parameter updates.

RSI Bench Relevance: Validates the parameter-free self-improvement path, crucial for real-time RSI loops. Reflective Metacognition milestone.
