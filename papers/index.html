RSI Bench Research Papers

  • HiMAC: Hierarchical Macro-Micro Learning Framework for Long-Horizon LLM Agents - Introduces a hierarchical learning framework where macro-level planning handles high-level goal decomposition and micro-level execution manages specific task steps for long-horizon tasks.
  • Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory (arXiv:2603.04257) - Introduces an indexed experience memory mechanism to scale long-horizon LLM agents.
  • AgentEvolver: Towards Efficient Self-Evolving Agent System (arXiv:2511.10395) - Introduces self-questioning, self-navigating, and self-attributing for efficient agent learning.
  • From Agentification to Self-Evolving Agentic AI (arXiv:2510.05596) - Outlines iterative self-improvement as the primary path for agent evolution.
  • EvoScientist: Towards Multi-Agent Evolving AI Scientists for End-to-End Scientific Discovery (arXiv:2603.08127) - Introduces an evolving multi-agent AI scientist framework with persistent ideation/experimentation memory for autonomous research strategy evolution.
  • AutoAdapt: An Automated Domain Adaptation Framework for LLMs (arXiv:2603.08181) - Proposes an automated framework for LLM domain adaptation using multi-agent debate and surrogate-based hyperparameter optimization.
  • Human-AI Collaboration for Scaling Agile Regression Testing: An Agentic-AI Teammate (arXiv:2603.08190) - Examines agentic-AI teammates in agile software development, advancing autonomous agent benchmarks in high-stakes human-collaboration scenarios.
  • AutoAgent: Evolving Cognition and Elastic Memory Orchestration for Adaptive Agents (arXiv:2603.09716) - Introduces a self-evolving multi-agent framework with evolving cognition and elastic memory, improving task success in dynamic environments.
  • YOLO-NAS-Bench: A Surrogate Benchmark with Self-Evolving Predictors for YOLO Architecture Search (arXiv:2603.09405) - Introduces a surrogate benchmark using self-evolving mechanisms to refine architecture search for detection.
  • MM-Zero: Self-Evolving Multi-Model Vision Language Models From Zero Data (arXiv:2603.09206) - Presents a multi-role, zero-data self-evolving RL framework (Proposer/Coder/Solver) for VLM reasoning.
  • RetroAgent: From Solving to Evolving via Retrospective Dual Intrinsic Feedback (arXiv:2603.08561) - Proposes an online RL framework with hindsight self-reflection for continuous agent evolution via intrinsic feedback and lesson distillation.
