Recursive Self-Improvement (RSI) Research Audit

Last Updated: 2026-04-01

Experiential Reflective Learning for Self-Improving LLM Agents

Authors: Victor Xing, et al.

Link: arXiv:2603.24639

Breakthrough: Reflecting on single-attempt experiences to extract transferable heuristics enables effective agent self-improvement without parameter updates.

Relevance to yanhua.ai RSI Bench: Validates the parameter-free self-improvement path, crucial for real-time RSI loops. Reflective Metacognition milestone.

SkillReducer: Optimizing LLM Agent Skills for Token Efficiency

Authors: Yudong Gao, Zongjie Li, et al.

Link: arXiv:2603.29919

Breakthrough: Two-stage optimization for skill routing and body structure, reducing token consumption by ~40% while improving functional quality.

Relevance to yanhua.ai RSI Bench: Critical for benchmarking attention efficiency and context management in long-horizon self-improving agents.

Beyond pass@1: A Reliability Science Framework for Long-Horizon LLM Agents

Authors: Aaditya Khanal, Yangyang Tao, Junxiu Zhou

Link: arXiv:2603.29231

Breakthrough: Introduces Reliability Decay Curve (RDC) and Meltdown Onset Point (MOP) to measure agent consistency on tasks of varying duration.

Relevance to yanhua.ai RSI Bench: Establishes "Reliability" as a first-class metric alongside capability for evaluating recursive agent evolution.

Improving Efficiency of GPU Kernel Optimization Agents using a Domain-Specific Language and Speed-of-Light Guidance

Authors: Siva Kumar Sastry Hari, et al.

Link: arXiv:2603.29010

Breakthrough: Compact DSL and Speed-of-Light (SOL) performance bounds to steer and budget search in GPU optimization tasks.

Relevance to yanhua.ai RSI Bench: Demonstrates the effectiveness of "First-Principles Guidance" in preventing search drift and diminishing returns in specialized RSI loops.

Polaris: A Gödel Agent Framework for Small Language Models through Experience-Abstracted Policy Repair

Authors: Aditya Kakade, Vivek Srivastava, Shirish Karande

Link: arXiv:2603.23129

Breakthrough: Implements a Gödel agent framework for small models that performs recursive self-improvement through "policy repair" and "experience abstraction."

Relevance to yanhua.ai RSI Bench: Direct metric for quantifying policy-level vs response-level evolution in self-improving loops.

SkillRL: Evolving Agents via Recursive Skill-Augmented Reinforcement Learning

Authors: Peng Xia, Jianwen Chen, et al.

Link: arXiv:2602.08234

Breakthrough: Distills raw experiences into a "SkillBank" for recursive evolution during RL, reducing token footprint while improving performance by 15.3%.

Relevance to yanhua.ai RSI Bench: Direct metric for measuring the efficiency gain of skill-based abstraction in self-improving loops.

MemRL: Self-evolving agents via runtime reinforcement learning on episodic memory

Authors: ArXiv 2601.03192

Link: arXiv:2601.03192

Breakthrough: Runtime reinforcement learning on episodic memory to enable continuous agent evolution during deployment.

Relevance to yanhua.ai RSI Bench: Establishes the Runtime Evolution paradigm, moving away from offline fine-tuning for RSI cycles.
