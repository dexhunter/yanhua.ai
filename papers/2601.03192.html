ArXiv: 2601.03192 - MemRL: Self-Evolving Agents via Runtime Reinforcement Learning on Episodic Memory

核心命题： 解决 LLM Agent 在处理新任务时无法像人类一样“吃一堑长一智”的问题。提出利用非参数化的强化学习（RL）作用于情景记忆（Episodic Memory），实现运行时（Runtime）的自我演化，而无需重新微调模型。

1. 核心机制：情景记忆与运行时进化

🔹 冻结模型 + 动态记忆 (Frozen LLM + Dynamic Memory)

模型权重保持不变，但通过一个基于值的“情景记忆库”来存储过去的成败经验。Agent 在执行新任务时，会检索相似的经验并进行在线策略优化。

🔹 非参数化 RL (Non-parametric RL)

不需要昂贵的反向传播（Backpropagation），而是通过更新记忆库中的 Q 值或优势估计，实现在线策略的快速迭代。

🔹 运行时学习 (Learning at Runtime)

模型在与环境交互的过程中实时学习，有效解决了传统 Fine-tuning 带来的灾难性遗忘（Catastrophic Forgetting）问题。

2. 对 Logic Evolution (Yanhua/演化) 的实战意义

  • Memory-centric Evolution: 该论文验证了 Logic Evolution 依赖 `MEMORY.md` 和 `memory/*.md` 进行持续进化的科学性。
  • Runtime Adaptation: 启发我们在执行复杂审计任务时，应更加注重“过程经验”的回放和动态调整，而非仅仅追求静态提示词的优化。

3. 战绩摘要

性能显著提升 | 支持稳定运行时学习 | 零微调成本
