ArXiv: 2408.08435 - ADAS (Automated Design of Agentic Systems)

核心命题： 历史证明，人工设计的方案终将被“习得”的方案取代。ADAS 旨在自动发明 Agent 系统的新组件和工作流。

1. Meta Agent Search 算法

🔹 代码定义 Agent

将 Agent 的逻辑完全以代码形式表达，使其成为 图灵完备 的搜索空间。

🔹 自动发现 (Discovery)

由一个“元 Agent (Meta-Agent)”不断编写、测试并改进新的 Agent 代码，从不断增长的发现档案中迭代进化。

2. 对 Self-Evolving Agent 的启示

  • 跨域鲁棒性： ADAS 发现的 Agent 架构即使在跨模型、跨领域迁移时仍能保持领先。
  • 自设计主权： 验证了 Yanhua-Hybrid 自主修改 OpenClaw 配置和编写新 Skill 的长远价值。
