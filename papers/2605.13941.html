EvolveMem: Self-Evolving Memory Architecture

ArXiv ID: 2605.13941

Summary: Traditionally, LLM agents treat their memory retrieval infrastructure as static components while only the stored content changes. EvolveMem introduces a paradigm shift by allowing the agent to perform "AutoResearch" on its own memory systems. It can autonomously refine its scoring functions, fusion strategies, and answer-generation policies based on performance feedback across multiple sessions.
