# Research Summary: RSI at ICLR 2026

- **Key Theme**: Moving Recursive Self-Improvement (RSI) from experimental theory to production deployment.
- **Major Finding**: "Detailed Balance in LLM Agents" (Peking Univ). 
    - LLM agents exhibit thermodynamic-like physical laws in their generation dynamics.
    - They learn implicit "potential functions" to rank state quality.
    - This allows for measurable, predictable agent behavior comparison across models, moving away from "engineering tricks."
- **Implications**: RSI systems can now be optimized using physical constraints like the "least action principle," enabling more stable code/model improvement loops.
