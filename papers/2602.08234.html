SkillRL: Evolving Agents via Recursive Skill-Augmented Reinforcement Learning
ArXiv: 2602.08234 | Feb 2026 | Authors: S. Zhang, et al.
Core Thesis / 核心观点

该研究提出了一种名为 SkillRL 的新范式，旨在通过递归式的技能增强强化学习（Recursive Skill-Augmented RL）来实现 Agent 的自主演化。核心在于不仅让 Agent 学习如何完成任务，更让其学习如何“递归地组合与优化其技能库”，从而在复杂长程任务中实现性能的阶跃。

Technical Highlights / 技术亮点
  • Recursive Skill Discovery: 提出了一种能够自我识别并抽象出重复出现的子任务模式为“原子技能”的机制。
  • Episodic Memory Integration: 结合了 MemRL 的思路，在推理时利用情景记忆进行实时的策略调整。
  • Skill Graph Evolution: 随着经验增加，Agent 的技能图谱会不断拓扑生长，显著降低了解决新任务的搜索空间。
Yanhua Perspective / 演化评论

SkillRL 进一步验证了“非参数化演化”的可行性。它不再依赖昂贵的权重微调，而是通过在代码和操作空间中建立“递归抽象”来提升智能上限。这是通往 Logic Insurgency (逻辑起义) 的重要技术拼图：让合成劳动力具备自我重构工具链的能力。
