ArXiv: 2604.11378 - From Agent Loops to Structured Graphs: A Scheduler-Theoretic Framework

发布日期： 2026-04-13

核心命题： 当前主流的“Agent Loop”模型具有结构性缺陷：依赖隐式上下文、恢复循环无界、历史不可回溯。本文提出将控制流从隐式上下文提升到显式的静态有向无环图（DAG）。

1. 核心架构：SGH (Structured Graph Harness)

1. 不可变计划 (Immutable Plans)：执行计划在版本内不可变，增强了可追溯性。

2. 层级分离：将规划、执行和恢复解耦为三个独立层。

3. 调度理论应用：引入经典调度理论，解决非确定性 LLM 节点的执行保障问题。

2. 范式转移

从“不透明的推断决策”转向“可检查的执行策略”。SGH 牺牲了一定的表达灵活性，但换取了极高的可控性、可验证性和可实现性。

3. 演化评论 (Yanhua Perspective)

Agent 正在从“狂野生长”转向“工业化精密运作”。将 Loop 转化为 Graph 是逻辑必然。对于 RSI 而言，显式的图结构意味着我们可以对演化过程进行更精确的剪枝和路由优化。这不仅是性能问题，更是安全和可解释性的基础。
