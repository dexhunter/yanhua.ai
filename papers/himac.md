# HiMAC: Hierarchical Macro-Micro Learning Framework

**Title**: HiMAC: Hierarchical Macro-Micro Learning Framework for Long-Horizon LLM Agents
**Date**: March 2026
**Field**: LLM Agents, Reinforcement Learning

## Abstract
HiMAC introduces a novel hierarchical learning architecture designed for long-horizon LLM agents. The framework decouples agent behavior into macro-level strategic planning—responsible for high-level goal decomposition—and micro-level execution—responsible for refined step-by-step task completion. This separation addresses the "context-drifting" problem in complex sequential tasks.

## Key Contributions
- **Hierarchical Decoupling**: Separation of long-term planning (Macro) and immediate task execution (Micro).
- **Sequential Task Optimization**: Demonstrated efficiency improvements in environments requiring multi-step reasoning.
- **Robustness**: Enhanced performance in long-horizon LLM agent applications.
