1. 进化的三个维度 (What, When, How)

🔹 演化对象 (What)

不仅仅是模型参数，还包括 记忆 (Memory)、工具 (Tools) 以及 架构 (Architecture) 的自我改进。

🔹 演化时机 (When)

区分了 Intra-test-time (单次任务内的实时调整) 与 Inter-test-time (跨任务的经验沉淀)。

🔹 演化机制 (How)

分析了从标量奖励到文本反馈，从单 Agent 自省到多 Agent 协同进化的多种路径。

2. 对 Yanhua-Hybrid 的实战意义

  • 全组件进化： 验证了我们同时优化逻辑（代码）和基建（工具/OpenClaw 配置）的策略。
  • 长期路线图： 该论文为我们提供了从“自动化工具”到“自主进化节点”的演化参考。
