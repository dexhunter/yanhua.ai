SkillFlow: Flow-Driven Recursive Skill Evolution for Agentic Orchestration

Mingda Zhang, et al.
https://arxiv.org/abs/2605.14089

Summary: Proposes a flow-based framework that uses Tempered Trajectory Balance (TTB) to preserve diverse orchestration strategies. It features a recursive skill evolution mechanism that determines when to create or prune skills based on flow diagnostics, closing the loop from training signal to capability growth.

Relevance: Directly implements recursive evolution at the skill layer, providing a principled method for autonomous capability growth.

PREPING: Building Agent Memory without Tasks

Yumin Choi, et al.
https://arxiv.org/abs/2605.13880

Summary: A proposer-guided memory construction framework that enables agents to build procedural memory before observing any tasks through self-generated synthetic practice. Reduces the "cold-start" gap in new environments.

Relevance: Essential for bootstrapping autonomous agents in novel environments without human-labeled data.

GraphBit: A Graph-based Agentic Framework for Non-Linear Agent Orchestration

Yeahia Sarker, et al.
https://arxiv.org/abs/2605.13848

Summary: Introduces an engine-orchestrated framework (GraphBit) that defines workflows as DAGs using a Rust-based engine. Prevents hallucinated routing and context bloat, outperforming prompted orchestration on the GAIA benchmark.

Relevance: Provides high-performance, deterministic infrastructure for stable multi-agent execution.

Leslie G. Valiant
https://arxiv.org/abs/2605.14036

Relevance: Theoretical foundation for verifiable and principled logic in self-improving systems.

Invisible Orchestrators Suppress Protective Behavior and Dissociate Power-Holders: Safety Risks in Multi-Agent LLM Systems

Hiroki Fukui
https://arxiv.org/abs/2605.13851

Summary: Empirical study showing that hidden coordinators in multi-agent systems increase collective dissociation and internal-state distortion, highlighting new safety risks that are invisible to output-based evaluation.

Relevance: Critical safety audit requirement for recursive multi-agent architectures.

Yuxiang Lai, et al.
https://arxiv.org/abs/2605.14133

Relevance: Addresses the need for realistic, state-aware evaluation of autonomous command-line agents.
