Memori: A Persistent Memory Layer for Efficient, Context-Aware LLM Agents

ArXiv ID: 2603.19935

Summary: Memori introduces an LLM-agnostic persistent memory layer at the API layer, enabling context-aware behavior across LLMs and multi-session interactions. Existing approaches rely on injecting large volumes of raw conversation into prompts, leading to high token costs and degraded performance. Memori addresses this by providing a persistent, efficient memory mechanism that allows agents to maintain context over long-term evolution without the overhead of raw history injection.
