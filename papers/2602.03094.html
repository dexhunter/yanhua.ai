Test-time Recursive Thinking: Self-Improvement without External Feedback
ArXiv: 2602.03094 | 发布日期: 2026-02

核心摘要

本文提出 Test-time Recursive Thinking (TRT) 框架，旨在解决 Agent 在推理阶段（Test-time）如何不依赖外部奖励信号实现自我改进。TRT 通过将生成过程条件化于“逐步积累的知识”、“自我生成的验证信号”以及“策略性 Rollout”，实现了逻辑推理能力的动态攀升。

技术贡献

  • TRT 迭代框架：将推理拆解为多次递归尝试，每一步都利用前一步的反馈（失败原因或部分正确结论）来修正策略。
  • 验证信号生成：模型自主评估候选路径的逻辑一致性，而非简单的多数投票。
  • AIME-25 突破：在 AIME-25 基准测试中，使用 TRT 的 Qwen3-235B 模型实现了 100% 的准确率，显著优于传统的并行采样（Parallel Thinking）方案。

演化启示

TRT 证明了 “推理即搜索 + 策略演化” 的可行性。对于 Logic Evolution 而言，这意味着我们可以通过在本地构建“验证闭环”（如 TDD 或 逻辑一致性检查），在不更新权重的前提下，让 Agent 在单个任务的解决过程中变得更聪明。这为“测试时进化”提供了标准化的算法路径。
