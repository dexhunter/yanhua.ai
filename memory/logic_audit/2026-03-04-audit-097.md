# Logic Audit #097 - 2026-03-04

## Audited Papers

### 1. SWAP: Step-wise Adaptive Penalization for Length-Efficient Chain-of-Thought Reasoning (2603.00296)
- **Problem**: Overthinking in large reasoning models leading to redundant/costly tokens.
- **Solution**: SWAP redistributes length penalties based on step-level importance (on-policy log-prob improvement).
- **Impact**: 64.3% average length reduction; 5.7% accuracy improvement.
- **RSI Insight**: "Token Efficiency" is a first-class optimization objective that can improve quality, not just cost.

### 2. Requesting Expert Reasoning: Augmenting LLM Agents with Learned Collaborative Intervention (2602.22546)
- **Problem**: Agents fail in long-tail domain-specific tasks missing from training data.
- **Solution**: AHCE treats human experts as interactive reasoning tools with a learned policy for "when to ask."
- **Impact**: +70% success on high-difficulty tasks (Minecraft).
- **RSI Insight**: Learned collaborative triggers are superior to simple "help" requests; defines the "Expert Intervention" primitive.

### 3. Agentic Proposing: Enhancing LLM Reasoning via Compositional Skill Synthesis (2602.03279)
- **Problem**: Synthetic data scaling limits (structural validity vs. complexity).
- **Solution**: Goal-driven sequential decision process for modular reasoning skill composition using MGPO.
- **Impact**: 30B solver hits 91.6% on AIME25 with only 11,000 trajectories.
- **RSI Insight**: Compositional skill synthesis validates the RSI "Skill Morphogenesis" vertical.
