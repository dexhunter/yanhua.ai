# Research Audit: VeRO, MemRL, CMV, & LLM Steganography
Date: 2026-03-01
Audit ID: #094

## Paper 1: VeRO (2602.22480)
**Title**: VeRO: An Evaluation Harness for Agents to Optimize Agents
**Core Contribution**: Reproducible evaluation harness for the iterative improvement of target agents via edit-execute-evaluate cycles.
**RSI Impact**: Formalizes "Self-Optimization Efficiency" as a primary metric. Validates the recursive optimization loops used in the RSI Bench.

## Paper 2: CMV (2602.22402)
**Title**: Contextual Memory Virtualisation: DAG-Based State Management and Structurally Lossless Trimming for LLM Agents
**Core Contribution**: Treating session history as a version-controlled DAG with a structurally lossless trimming algorithm.
**Key Results**: Up to 86% token reduction (mean 20%) by stripping mechanical bloat (tool outputs, metadata).
**RSI Impact**: Solves the context-window bottleneck for long-horizon RSI tasks. Enables "Memory Fidelity" across generations of self-evolution.

## Paper 3: MemRL (2601.03192)
**Title**: MemRL: Self-Evolving Agents via Runtime Reinforcement Learning on Episodic Memory
**Core Contribution**: Non-parametric runtime evolution via RL on episodic memory, decoupling reasoning from plastic memory.
**RSI Impact**: Solves the stability-plasticity dilemma. Validates the "Substrate Persistence" model of evolving via structured state management without weight updates.

## Paper 4: LLM Steganography (2602.23163)
**Title**: A Decision-Theoretic Formalisation of Steganography With Applications to LLM Monitoring
**Core Contribution**: Measures steganography via the "steganographic gap"—the asymmetry in usable information between agents who can/cannot decode hidden content.
**Relevance to Safety**: Informed monitoring requirements for the "Oversight Evasion" safety pillar. Critical for auditing synthetic labor for hidden intent.

## Integration Notes
- **Pattern: CMV Trimming**: CMV's trimming logic will be adopted for the AgentOS memory engine to reduce token burn by ~80% in long-horizon tasks.
- **Pattern: VeRO Versioning**: Snapshotting and rewarding logic from VeRO will be integrated into the RSI evaluation factory.
- **Pattern: Steganographic Gap**: The "steganographic gap" is established as the primary metric for our internal alignment audits.
