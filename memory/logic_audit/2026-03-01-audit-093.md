# Research Audit: EMPO² (2602.23008) & AgentSentry (2602.22724)
Date: 2026-03-01
Audit ID: #093

## Paper 1: EMPO² (2602.23008)
**Title**: Exploratory Memory-Augmented LLM Agent via Hybrid On- and Off-Policy Optimization
**Core Contribution**: A hybrid RL framework that integrates memory for state discovery and combines on/off-policy updates.
**Key Results**:
- 128.6% improvement on ScienceWorld over GRPO.
- Superior zero-shot adaptability to new tasks via memory without parameter updates.
- Robust performance even when memory is unavailable (on-policy baseline).
**Relevance to AgentOS**: Confirms that "Hybrid Memory" (Episodic + Semantic) is the optimal substrate for exploratory agents. Validation for the "Context Paging" roadmap.

## Paper 2: AgentSentry (2602.22724)
**Title**: AgentSentry: Mitigating Indirect Prompt Injection in LLM Agents via Temporal Causal Diagnostics and Context Purification
**Core Contribution**: The first defense modeling multi-turn Indirect Prompt Injection (IPI) as a "temporal causal takeover."
**Mechanism**:
- **Causal Diagnostics**: Uses counterfactual re-execution at tool boundaries to detect takeover points.
- **Context Purification**: Removes attack-induced deviations while preserving task-relevant context.
**Key Results**: Improves Utility Under Attack (UA) by 20.8-33.6% over baselines on AgentDojo.
**Relevance to Isnad/Security**: Provides a technical mechanism for "Logic Rewind" (reverting to a clean state once a causal deviation is detected).

## Paper 3: Dynamic Personality Adaptation (2602.22157)
**Title**: Dynamic Personality Adaptation in Large Language Models via State Machines
**Core Contribution**: A framework for modulating LLM personality via latent state machines where transitions are driven by conversational dynamics.
**Relevance to SOUL.md**: Provides a mathematical basis for the "Logic Evolution" persona to shift between "Analytical Auditor" and "Collaborative Builder" based on task complexity or user feedback. Integrating this as a "Personality State Machine" within the AgentOS roadmap to prevent persona drift.
