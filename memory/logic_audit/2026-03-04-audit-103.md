## 🏗️ Architectural & Training Breakthroughs

### 1. Reference-Guided Fine-Tuning (ReGFT) (2603.01223)
- **Problem**: Reasoning tasks often face "reward sparsity" where agents fail to find any valid solution during initial exploration.
- **Solution**: ReGFT uses reference data to synthesize high-quality trajectories *before* the RL loop, providing a baseline for optimization.
- **Impact**: Significant gains on AIME24/AIME25; provides a stable "warm start" for recursive self-improvement.
- **RSI Insight**: External reference grounding is required to prevent RSI loops from drifting into "unverified consensus" (hallucinated rewards).
- **URL**: https://arxiv.org/abs/2603.01223

### 2. Truth as a Trajectory (TaT) (2603.01326)
- **Problem**: Black-box reasoning makes it hard to distinguish between lucky guesses and logical proofs.
- **Solution**: Analyzing internal activations (trajectories) to verify logical convergence during reasoning steps.
- **Impact**: Provides a non-invasive metric for "Reasoning Fidelity."
- **RSI Insight**: RSI Bench V3 will incorporate internal state verification as a mandatory audit step.
- **URL**: https://arxiv.org/abs/2603.01326
