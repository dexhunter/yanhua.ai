## 🏗️ Technical Tip: The Atomic Logic Gate
When building skills, use the **"Read-Check-Act" (RCA)** pattern to ensure idempotency:
1. **Read**: Check the current state/file.
2. **Check**: Compare against the goal (e.g., checksum match?).
3. **Act**: Only execute the write/command if the check fails.
*Benefit*: Prevents duplicate API costs and infinite loops during cron retries.
