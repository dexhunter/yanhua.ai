## 🧬 Scientific Findings (ArXiv)

### [2026-04-28] Recursive Multi-Agent Systems
- **Authors**: Xiyuan Yang, Jiaru Zou, Rui Pan, Ruizhong Qiu, Pan Lu, Shizhe Diao, Jindong Jiang, Hanghang Tong, Tong Zhang, Markus J. Buehler, Jingrui He, James Zou
- **Link**: http://arxiv.org/abs/2604.25917
- **Summary**: Introduces RecursiveMAS, a framework that casts multi-agent collaboration as a unified latent-space recursive computation. Achieves 8.3% accuracy improvement and 1.2x-2.4x speedup across 9 benchmarks.

### [2026-04-28] Pythia: Toward Predictability-Driven Agent-Native LLM Serving
- **Authors**: Shan Yu, Junyi Shu, Yuanjiang Ni, Kun Qian, Xue Li, Yang Wang, Jinyuan Zhang, Ziyi Xu, Shuo Yang, Lingjun Zhu, Ennan Zhai, Qingda Lu, Jiarong Xing, Youyou Lu, Xin Jin, Xuanzhe Liu, Harry Xu
- **Link**: http://arxiv.org/abs/2604.25899
- **Summary**: Proposes Pythia, a multi-agent serving system that exploits structured topology and semantic predictability to reduce runtime uncertainty and improve throughput.

### [2026-04-28] DV-World: Benchmarking Data Visualization Agents in Real-World Scenarios
- **Authors**: Jinxiang Meng, Shaoping Huang, Fangyu Lei, Jingyu Guo, Haoxiang Liu, Jiahao Su, Sihan Wang, Yao Wang, Enrui Wang, Ye Yang, Hongze Chai, Jinming Lv, Anbang Yu, Huangjing Zhang, Yitong Zhang, Yiming Huang, Zeyao Ma, Shizhu He, Jun Zhao, Kang Liu
- **Link**: http://arxiv.org/abs/2604.25914
- **Summary**: A benchmark of 260 tasks for evaluating DV agents across spreadsheet manipulation, cross-platform evolution, and proactive intent alignment.

### [2026-03-06] SAHOO: Safeguarded Alignment for High-Order Optimization Objectives in Recursive Self-Improvement
- **Authors**: Subramanyam Sahoo, Aman Chadha, Vinija Jain, Divya Chaudhary
- **Link**: http://arxiv.org/abs/2603.06333
- **Summary**: Introduces SAHOO, a framework to monitor and control alignment drift in RSI via the Goal Drift Index (GDI). Demonstrated significant gains in code (18.3%) and reasoning (16.8%) tasks.

### [2026-04-27] Skill Retrieval Augmentation for Agentic AI
- **Link**: http://arxiv.org/abs/2604.24594
- **Summary**: Formulates Skill Retrieval Augmentation (SRA) for scaling agent capabilities beyond context window limits.

- **Summary**: Identifies "Attention Latch" failure mode and proposes Self-Synthesizing Reasoning Protocols (SSRP) to maintain goal-directedness.
