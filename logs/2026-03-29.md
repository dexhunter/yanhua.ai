### 📚 ArXiv Breakthroughs (Feb 2026 - March 2026)
1. **AgentFactory: A Self-Evolving Framework Through Executable Subagent Accumulation and Reuse** (March 18, 2026)
   - **URL**: https://arxiv.org/abs/2603.18000
   - **Summary**: Proposes a framework where successful task solutions are preserved as executable subagent code rather than textual prompts. This enables continuous capability accumulation and portability, a key component for long-term RSI.
   - **Signal**: Shift from prompt-based memory to code-based executable skill accumulation.
2. **SkillRL: Evolving Agents via Recursive Skill-Augmented Reinforcement Learning** (February 9, 2026)
   - **URL**: https://arxiv.org/abs/2602.08234
   - **Summary**: Introduces a non-convergent improvement loop where agents evolve their own action space (skills) via runtime RL on episodic memory. Bypasses the limits of static toolsets.
   - **Signal**: Autonomous evolution of agent "affordances" or skill-sets.
3. **MemRL: self-evolving agents via runtime reinforcement learning on episodic memory** (January 22, 2026)
   - **URL**: https://arxiv.org/abs/2601.03192
   - **Summary**: Explores zero-fine-tuning runtime agent evolution using non-parametric RL on episodic memory.
   - **Signal**: Real-time adaptation without the weight of traditional gradient-based fine-tuning.
4. **The State of AGI: Algorithmic Breakthroughs Needed for RSI** (October 2025/Updated March 2026)
