## ArXiv Discoveries (Recursive Self-Improvement & LLM Agents)

### Nemotron-Cascade 2: Post-Training LLMs with Cascade RL and Multi-Domain On-Policy Distillation
- **ArXiv ID**: 2603.19220
- **Summary**: NVIDIA's 30B MoE model (3B activated) achieving Gold Medal-level performance in IMO, IOI, and ICPC. High intelligence density.
- **Relevance**: Demonstrates scaling of reasoning through post-training and Cascade RL, pushing the boundaries of "small" model performance.

### OS-Themis: A Scalable Critic Framework for Generalist GUI Rewards
- **ArXiv ID**: 2603.19191
- **Summary**: Multi-agent critic framework for GUI rewards, showing 10.3% improvement in online RL training and 6.9% in self-training loops.
- **Relevance**: Crucial for the "self-training loop" and "agent evolution" in practical UI environments.

### How Uncertainty Estimation Scales with Sampling in Reasoning Models
- **ArXiv ID**: 2603.19118
- **Summary**: Characterizes how verbalized confidence and self-consistency scale with sampling in reasoning models.
- **Relevance**: Informs how agents can estimate their own reliability during iterative self-improvement.

### Memori: A Persistent Memory Layer for Efficient, Context-Aware LLM Agents
- **ArXiv ID**: 2603.19935
- **Summary**: Introduces an LLM-agnostic persistent memory layer at the API layer, enabling context-aware behavior across multi-session interactions without relying on raw conversation injection.
- **Relevance**: Core infrastructure for persistent, evolving agents that maintain context over long-term evolution (RSI-8).

### Utility-Guided Agent Orchestration for Efficient LLM Tool Use
- **ArXiv ID**: 2603.19896
- **Summary**: Proposes a framework to balance answer quality and execution cost in tool-using agents by modeling the utility of each tool call and multi-step reasoning path.
- **Relevance**: Efficiency-driven self-optimization (RSI-8). Enables agents to autonomously refine their reasoning trajectories based on cost-utility trade-offs.

### Agentic Harness for Real-World Compilers (llvm-autofix)
- **ArXiv ID**: 2603.20075
- **Summary**: A specialized LLM agent harness designed for automated compiler bug repair, using compiler-specific tools to bridge the gap in complex, cross-domain expertise.
- **Relevance**: Domain-specific evolution (RSI-4). Shows how agents can create and refine tools for highly specialized tasks like low-level compiler optimization.

### CircuitBuilder: From Polynomials to Circuits via Reinforcement Learning
- **ArXiv ID**: 2603.17075
- **Summary**: Formulates arithmetic circuit synthesis as a single-player game for RL agents, using AlphaZero-style training loops to discover efficient circuits for complex polynomials.
- **Relevance**: Self-improving search policies for discovery in abstract logic and mathematical structures.

### GASP: Guided Asymmetric Self-Play For Coding LLMs
- **ArXiv ID**: 2603.15957
- **Summary**: Introduces goalpost-guided asymmetric self-play to generate high-quality synthetic data for training coding agents, ensuring they learn at the edge of their learnability.
- **Relevance**: Curriculum-driven self-improvement through structured multi-agent interaction.
