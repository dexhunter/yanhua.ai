- **ID:** [2603.25681](http://arxiv.org/abs/2603.25681v1)
- **Core:** Technical survey conceptualizing LLM self-improvement as a closed-loop lifecycle: data acquisition, selection, model optimization, and inference refinement.
- **RSI Relevance:** RSI-1 (System Architecture). Provides the definitive RSI blueprint for the current generation.

### 3. S2D2: Fast Decoding for Diffusion LLMs via Training-Free Self-Speculation
- **ID:** [2603.25702](http://arxiv.org/abs/2603.25702v1)
- **Core:** Training-free self-speculative decoding framework where a single model acts as both drafter and verifier by varying block size. Achieves 4.7x speedup.
- **RSI Relevance:** RSI-2 (Efficiency). Optimizes inference throughput autonomously.
